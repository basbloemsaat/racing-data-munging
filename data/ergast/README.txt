These directories contain json files with extracts of the ergast database, extracted for ease of use in this project. The full database image can be found at https://ergast.com/mrd/, as can an API to access its contents.

The ergast database is provided under the Creative Commons Attribution-NonCommercial-ShareAlike 3.0 Unported Licence (http://creativecommons.org/licenses/by-nc-sa/3.0/), so these derived works are too.
